Introduction

DataToolkitCommon takes the skeleton provided by DataToolkitCore and builds the basic functionality needed to make DataToolkit actually useful.

To gain an understanding of how these two components fit together, it is recommended the DataToolkitCore docs be looked at first to understand the system, and then these docs for useful transformers and plugins to actually work with your data.

Note that in addition to being displayed here, you can also view the documentation of the various drivers in the data> REPL like so:

data> ?: # See a list of all known drivers
 storage transformers (<count>)
   [...]
 loader transformers (<count>)
   [...]
 writer transformers (<count>)
   [...]

data> ?:csv # View the CSV loader documentation
  Parse and serialize CSV data
  [...]
